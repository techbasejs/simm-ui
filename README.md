- @simmx/code-highlighter

- @simmx/demo

### For development

Install nodejs with nvm

```sh
nvm use
```

Install dependencies

```sh
pnpm install
```

Build all packages

```sh
pnpm build all
```

Run storybook

```sh
pnpm storybook
```

Serve a docs

```
pnpm docs:dev
```
