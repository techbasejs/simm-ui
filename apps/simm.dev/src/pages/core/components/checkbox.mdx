Checkboxes allow users to select multiple items from a list of individual items, or to mark one individual item as selected.

## Accessbility

- Built with a native HTML `<input>` element.
- Full support for browser features like form autofill.
- Keyboard focus management and cross browser normalization.

## API

### Checkbox Props

| Attribute        | Type                                                                         | Description                                                  | Default   |
| ---------------- | ---------------------------------------------------------------------------- | ------------------------------------------------------------ | --------- |
| label            | `ReactNode`                                                                  | The label of checkbox.                                       | -         |
| variant          | `filled` \| `outlined`                                                       | The button appearance style.                                 | `filled`  |
| color            | `default` \| `primary` \| `success` \| `warning` \| `error`                  | The button color theme.                                      | `default` |
| size             | `sm` \| `md` \| `lg`                                                         | The button size.                                             | `md`      |
| disabled         | `boolean`                                                                    | Whether the button is loading.                               | `false`   |

### Checkbox Events

Accept all events of `<input type="checkbox" />`
